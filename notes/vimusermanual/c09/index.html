usr_09 使用 GUI 版本

09.1 GUI 版本的组件

gvim <FILENAME>
vim -g <FILENAME>

标题栏文件名后可能会跟一个符号

- 文件不能被修改

+ 已经被修改过

= 文件只读

=+ 只读但被修改过

没有标记则是一个普通的打开但没修改过的文件。

09.2 使用鼠标

使用鼠标可以移动光标和选中文本等

09.3 剪贴板

注意区分当前选区和真剪贴板。当前选区是可视模式框选的内容。

黏贴当前选区内容

"*P

注意大写 P 拷贝到光标前。

"*

寄存器对应当前选区

"+

寄存器对应真剪贴板。

09.4 选择模式

启用选择模式

:set selectmode+=mouse

在选择模式下，新输入内容将覆盖选中的部分。
