usr_11 从崩溃中恢复

11.1 基本恢复

一般来说如果硬盘没坏，崩溃了之后，文件的大部分内容可以恢复。

vim -r <FILENAME>

这时 Vim 会读取 .swp 文件。为安全起见可以另存这个新文件。

可以用 vimdiff 比较新旧文件。

如果编辑的时候没有文件名，可以进入对应的目录，然后

vim -r ""

11.2 交换文件在哪

Vim 可以把交换文件放在不同的地方，通常是原文件所在的目录。

可以进入目录，然后使用

vim -r

列出所有的交换文件。

11.3 是不是崩溃了？

有时不是崩溃，比如其他进程也在编辑文件，则在打开文件时，Vim 也会弹出警告。
