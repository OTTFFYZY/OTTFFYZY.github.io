OTTFF 的新 Blog

很久之前我有一个 xxdn 的 Blog，但是随着自己越来越懒其广告等不重要的信息越来越多，于是弃坑多年。

读到一些大牛自建的博客之后，也萌生了自己搭建一个界面简单干净而又功能强大的 Blog 的想法。

经过研究，大概两年前确立了利用 Github Pages 搭建静态博客的方案

常见的静态博客框架有

  • Github Pages 原生的 Jekyll
  • 基于 Node.js 的 Hexo
  • 基于 Golang 的 Hugo

Hugo 号称渲染最快，于是最终选择了 Hugo 的方案。

在两年挂机之后，终于在这个实习结束，尚未开学的闲暇时光重新开启了这项计划。

克服困难读完了 Hugo Doc

魔改了 Hugo-Primer 主题之后，自己定制化的 Blog 终见天日。

庆祝这个时刻！
